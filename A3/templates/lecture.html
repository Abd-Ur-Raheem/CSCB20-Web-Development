Lecture Notes

Week 1

Overview

A quick speed-refresher of Python. We go over variables, functions, string methods, lists, loops, and basic I/O. The material covered today will be especially relevant to those who have not taken one of A08, A48, or A20





Week 2

Overview

Relational algebra. In this tutorial, we go over relational algebra and writing relational algebra to perform queries. We also introduce SQLite and basic SQL concepts.





Week 3

Overview

For this week, we will be extracting actual data out of databases so you will be required to have SQLite in order to complete the lab.





Week 4

Overview

This week is dedicated to SQL case, subqueries, intro to webdev, frontend/backend and html





Week 5

Overview

This week is a full introduction to webdevelopement. We will learn about servers, webpages, browsers and frameworks

We will also be gettin started with HTML and CSS and creating our own webpages!





Week 6

Overview

This week we have an online lecture since we have our midterm on lecture day.

We will go more in depth with HTML, CSS and even touch on some Javascript. We will also learn some css styling and css grid/flexbox
